# VirtualBox Networking: NAT vs Internal Network Troubleshooting

## Overview

- **Adapter 1:** NAT for Internet access
- **Adapter 2:** Internal Network for isolated VM-to-VM communication

The intended network design was:

| Machine | NAT Network | Internal Homelab Network |

| Ubuntu Server | `10.0.2.15` | `10.0.0.10` |
| Ubuntu Desktop | `10.0.2.15` | `10.0.0.20` |
| Kali Linux | `10.0.2.15` | `10.0.0.30` |
