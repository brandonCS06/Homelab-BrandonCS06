# Homelab-BrandonCS06
A versatile, lightweight virtual environment running on Oracle VM VirtualBox. Configured with three distinct virtual machines to safely sandbox and test networking concepts.

## Purpose

This homelab is used to practice networking and Linux administration.

Current Environment

- Ubuntu Server
- Kali Linux
- Windows 11 Client (Planned)

Host Machine

- Intel i5-13400F
- 16 GB RAM
- VirtualBox

Goals

- Practice IP addressing
- Configure DHCP and DNS
- Test routing
- Capture packets with Wireshark
- Practice SMB and SSH
- Learn Active Directory
- Simulate enterprise networking

Current Status

| Machine | Status |
|---------|--------|
| Ubuntu Server | Complete |
| Kali Linux | Complete |
| Windows Client | Planned |
